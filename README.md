# FunWithTrackBall
Fun With blackberry trackball breakout board 
Connection:
BreakOut Board  to Arduino
BTN                Pin2 
RHT                Pin3
LFT                Pin4
UP                 Pin5
WHT                Pin6 PWM
DWN                Pin7 
GRN                Pin9 PWM
RED                Pin10PWM
BLU                Pin11PWM
